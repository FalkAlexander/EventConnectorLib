Welcome to EventConnectorLib's Documentation!
=============================================

EventConnectorLib is a comprehensive library designed to manage low-level event handling in the ZKMS event system. It facilitates connecting to the broker, setting up clients, and sending and receiving events, making it straightforward to implement modules within the system. 

Key Features
------------

- **Seamless Integration**: Easily connect to the broker and manage events.
- **Threaded Processing**: Efficient event handling with threaded queues.
- **Custom Event Handlers**: Define custom functions to manage incoming events.
- **Embedded HTTP Server**: Listen and process events through an internal HTTP server.
